Good to know

Frequently asked

Is Neoma really free?

Yes — every platform, no paid tier. Neoma is open source under the AGPL-3.0 licence, so you can read, modify and self-host it.

Where are my notes stored?

On your device. Point Neoma at a local folder and every page is a plain .md file you can open in any editor. In the browser without a folder, notes are kept in local storage on that machine. Nothing is uploaded anywhere.

Does it work offline?

Completely. Neoma is a local-first app and an installable PWA — it launches and runs with no network connection, and never phones home.

What's the difference between the web app and the desktop app?

Same app, same features. The desktop build adds a native window, system tray, launch-on-startup and native file dialogs. The web app needs no install and is the fastest way to try Neoma.

Can I move my notes to another tool later?

Always. They're standard Markdown files with no proprietary wrapper — there's nothing to export, because they were never locked in to begin with.
